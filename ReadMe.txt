要件

・Words_.jsonファイルには50個の英単語と、その英単語の意味と例文が格納されている。

・EnglishWords.htmlは、Words_01.jsonやWords_02.jsonなどナンバリングしていき、複数のjsonを同時に読み込むことが可能なHTML
・読み込んだファイルを、読み込み除外する機能を持つ

・EnglishWords.htmlは、チェックリストにいれた単語から新たにCheckWords_01.jsonなどナンバリングされたjsonを作成できる機能がある

・EnglishWords.htmlは意味の4択テスト機能を持ち、間違った単語を自動でチェックリストに入れ、意味と品詞と例文を表示する機能がある（押し間違えの場合、取り消す機能を持つ）
・4択テスト機能は、正解の場合は自動で次の問題へ進む
・4択テストで行う問題数を任意に設定できる
・4択テストはチェックリスト内の単語に絞る、チェックリスト外の単語に絞ることができる

・EnglishWords.htmlは任意の単語をチェックリストに追加できる

ユーザーの使用方法
・GeminiなどのAIに、英単語力を推定してもらい、覚えるべき単語を50個、意味、品詞、例文としてjsonで出力してもらう。
